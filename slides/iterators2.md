# For-of

ES6 implementa esta nueva forma de loop para objetor **Iterables**:

<code><pre>
const arr = [1, 2, 3, 4];
for (let item of arr) {
  console.log(item); // 1 // 2 // 3 // 4
}
</pre></code>


<code><pre>
const str = "Hola";
for (let c of str) {
  console.log(c); // H // o // l // a
}
</pre></code>
