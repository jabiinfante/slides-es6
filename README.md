Referencia de las nuevas funcionalidades que encontramos en ECMAScript 2015 o ES6.

Se cubren los siguientes aspectos:
 - let / const
 - rest
 - spread
 - destructuring
 - cadenas
 - iteradores
 - generadores
 - clases
 - modulos
 - promesas
